    How it works

    1. Run Blaze web + API locally (npm run dev:all from repo root).
    2. Log in via Open notepad — desktop syncs your session automatically.
    3. Approve a handoff action in Blaze.
    4. Desktop delivers the handoff file and opens Cursor.
