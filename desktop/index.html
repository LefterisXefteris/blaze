    How it works

    1. Run Blaze web + API locally (or point at production).
    2. Log in via Open Blaze and approve a handoff action.
    3. Desktop polls for confirmed handoffs and writes them locally.
    4. Cursor opens with the handoff file + rules snippet.

    Get your access token: log into Blaze in the browser, open DevTools → Application → Cookies → copy the Supabase access_token from the auth cookie, or use the JWT from your session.
